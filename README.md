# Amazon Sales Analysis Dashboard

## Overview
This project analyzes e-commerce sales data to uncover insights on revenue, profit, and customer behavior.

---

## Tools Used
- Python (Pandas)
- Power BI

---

## Dashboard Features
- Total Sales & Profit tracking
- Sales by Category
- Monthly Sales Trends
- Customer Segmentation
- Region-wise Sales Analysis

---

## Key Insights
- Electronics generated highest revenue
- Metro cities dominated sales
- UPI was most used payment method

---

## Outcome
Developed an interactive dashboard to support business decision-making.
